Smoke Test Report

======================================================================================================
To view consolidated report for smoke tests

Click on "SMOKE_REPORT.html"
This should open up a consolidated report with "Statistics by Tag" and "Statistics by Suite"
Click on any tag to drill down into test details

======================================================================================================
To view build information

Click on "SMOKE_REPORT.html"
Click on BUILD_INFO_STG/BUILD_INFO_TST under "Statistics by Tag"
This should open up test details page.
Click on the link under "Name" column
It should navigate to the test case "Scenario: Get Deployed Build Information on Staging"
Expand the Keyword "BuiltIn.Log ABOUT BUILD ON STAGING - ${build_info}, HTML"
This should display the build information
